문제를 이해하는 것도 사실 조금 힘들었는데,,, 일단 주어진 문자열 배열을 숫자로 바꿔서 그 숫자를 묶어서 개수를 세고자 생각함.
예시 : ["a","bc","d","efg","hi"] -> [1, 2, 1, 3, 2] -> 1이 2개 2개 2개 3이 1개
일단 String[] 을 int[]로 바꿨는데 막상 이 숫자를 어떻게 세지 고민을 하였고, 검색의 도움을 받아
길이별 개수를 저장할 배열을 만들어 arr[]을 돌면서 count[]에 값을 쌓고 거기서 가장 큰 값을 찾아내 반환

## 💻 코드

```java
class Solution {
    public int solution(String[] strArr) {
        int[] arr = new int[strArr.length];
        
        for (int i = 0; i < strArr.length; i++){
            arr[i] = strArr[i].length();   
        }
        
        int[] count = new int[31];
        
        for (int i = 0; i < arr.length; i++){
            count[arr[i]]++;
        }
        
        int max = 0;
        
        for (int i = 0; i < count.length; i++){
            if(count[i] > max){
                max = count[i];
            }
        }
        
        return max;
    }
}
```
