## 💡 문제 해결 전략

<!-- 문제를 어떻게 접근했는지, 어떤 알고리즘/자료구조를 사용했는지 적어주세요 -->
<!-- 그림이나 예시를 들어 설명하면 나중에 복습할 때 좋습니다 -->

배열을 주었고, 배열의 값이 1이면서 idx보다 크거나 같은 인덱스를 찾으라고 하였으니 
배열을 순회하여 이 두 가지의 조건을 맞는 것을 찾으면 되겠다고 생각하여 for문을 작성 

## 💻 코드

```java
class Solution {
    public int solution(int[] arr, int idx) {

        for(int i = 0; i < arr.length; i++){
            if(arr[i] == 1 && i >= idx){
                return i;
            }
        }
        
        return -1;
    }
}
```

내가 짠 코드의 경우 모든 배열 값을 다 확인하는 반면에, 이미 문제에서 idx 이상인 인덱스라고 얘기를 했으므로
i를 idx로 정해놓으면 반복 횟수가 줄어들기 때문에 더 좋은 코드라는 생각이 듦
