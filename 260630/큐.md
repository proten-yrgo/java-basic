일단 배열을 만들어 큐는 앞에 집어넣고 앞부터 뺀다고 생각하고 값을 넣으려고 생각함. (줄세우기)
큐의 가장 처음 자리를 front 끝 자리를 rear로 정하고,
삭제연산만 되는 곳이 front 삽입연산만 되는 곳이 rear 라고 할 때
값을 빼면(pop) front를 증가시키고 값을 넣으면(push) rear를 증가시킴

## 💻 코드

```java
package study;

import java.util.Scanner;

public class Queue {
    public static int[] queue;
    public static int front; //맨 앞
    public static int rear; //맨 뒤

    public static void main(String[] args) {
        queue = new int[15];

        push(1);
        push(2);
        front();
        back();
        size();
        empty();
        pop();
        pop();
        pop();
        size();
        empty();
        pop();
        push(3);
        empty();
        front();
    }

    static void push(int data1) {
        queue[rear] = data1;
        rear++; //다음에 넣을 곳 준비
    }

    static void pop() {
        if (front == rear) { //아무 값도 없을 때는 둘은 동일
            System.out.println("-1");
        } else {
            System.out.println(queue[front]);
            front++; //pop은 빠져나간 거니 맨 앞 값의 자리를 그 다음으로 변경
        }
    }

    static void front() {
        if (front == rear) {
            System.out.println("-1");
        } else {
            System.out.println(queue[front]);
        }
    }

    static void back() {
        if (front == rear) {
            System.out.println("-1");
        } else {
            System.out.println(queue[rear - 1]); 
			//rear는 값이 들어오고 나서 다음 들어오는 값을 넣기 위해 ++을 하기 때문에 실질적으로 마지막에 있는 값은 rear-1 의 자리에 있음
        }
    }

    static void size() {
        System.out.println(rear - front); 
		//값이 들어오면 rear가 증가하고 만약 pop을 하면 front가 증가하기 때문에 존재하는 값의 개수를 보려면 rear-front
    }

    static void empty() {
        if (front == rear) {
            System.out.println("1");
        } else {
            System.out.println("0");
        }
    }
}

```
