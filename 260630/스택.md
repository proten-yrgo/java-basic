스택은 나중에 들어온 값이 바로 나가는 즉, 한 방향만 생각하면 됨 (쌓기)
그러니 top만 존재하면 되고 값이 들어오면 top을 증가시킴 나가면 top을 감소시키면 됨


## 💻 코드

```java
package study;

public class stack {
    public static int[] stack;
    public static int top;

    public static void main(String[] args) {
        stack = new int[15];

        push(1);
        push(2);
        top();
        size();
        empty();
        pop();
        pop();
        pop();
        size();
        empty();
        pop();
        push(3);
        empty();
        top();
    }

    static void push(int data1) {
        stack[top] = data1;
        top++; //값이 들어왔으니 다음자리
    }

    static void pop() {
        if (top == 0) {
            System.out.println("-1");
        } else {
			System.out.println(stack[top]-1); //값이 들어오면서 top의 수는 다음자리로 넘어가니 실질적으로 값은 -1 자리에 있고
            top--; // 값을 뺐으니 top을 줄여줌 
        }
    }

    static void top() {
        if (top == 0) {
            System.out.println("-1");
        } else {
            System.out.println(stack[top-1]);
        }
    }

    static void size() {
        System.out.println(top);
    }

    static void empty() {
        if (top == 0) {
            System.out.println("1");
        } else {
            System.out.println("0");
        }
    }
}

```
