# 📌 문제 이름

> 예: 피자 나눠 먹기 (1)

## 🔗 문제 링크

- [문제 바로가기](https://school.programmers.co.kr/learn/courses/30/lessons/120814)

## 💡 문제 해결 전략

if문으로 접근하려고 함.
n이 7보다 작거나 같을 때 answer = 1로 정하고 else if 를 n이 7보다 클 때로 했는데 여기서 계산이 잘 안돼서 
검색을 했고 return (n + 6) / 7; 해당 식을 발견함
만약 기존 if문으로 해결했다면
if(n%7==0){
                answer = n/7;
            } else {
                answer = n/7+1;
            }
이렇게 했을 듯


## 💻 코드

```java
// class Solution {
    public int solution(int n) {
        return (n + 6) / 7;
    }
}
}
```
