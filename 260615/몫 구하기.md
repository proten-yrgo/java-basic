## 💡 문제 해결 전략

answer이 이미 선언되어 있고, return answer을 하고 있기 때문에 answer에 계산 값을 담아서 바로 리턴함

## 💻 코드

```java
// class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        
        answer = num1 / num2;
        
        return answer;
    }
}
```
