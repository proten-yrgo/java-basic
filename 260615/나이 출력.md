# 📌 문제 이름

> 예: 나이 출력

## 🔗 문제 링크

- [문제 바로가기](https://school.programmers.co.kr/learn/courses/30/lessons/120820)

## 💡 문제 해결 전략

나이는 태어나서부터 1살 이기 때문에 현재 연도에서 나이를 빼고 1을 더해주었습니다.

## 💻 코드

```java
// class Solution {
    public int solution(int age) {
        return 2022 - age + 1;
    }
}
```
