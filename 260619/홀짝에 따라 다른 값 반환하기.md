일단 n의 홀짝을 판단해야되기 때문에 짝수로 나누었을 때 나머지가 0이거나 1인 것으로 판단하고자 함.
그래서 if문을 만들어야겠다고 생각하였고, n 이하의 경우에서 또 홀짝에 맞는 계산이 들어가야하기 때문에
안에 for문을 넣고자 생각함. i가 주어진 n만큼 돌 때 i의 홀짝을 판별하여 그에 맞게끔 계산을 하려고 안에 if문을 넣음
그리고 해당 값을 result에 넣어 반환함.

## 💻 코드

```java
// 코드를 여기에 붙여넣어주세요 (주석 포함 권장)
class Solution {
    public int solution(int n) {
        int result = 0;
        
        if(n % 2 == 0){
            for (int i = 1; i <= n; i++){
                if(i % 2 == 0){
                    result += i*i;
                }
            }
        } else {
            for (int i = 1; i <= n; i++){
                if(i % 2 != 0){
                    result += i;
                }
            }
        }
        
        return result;
    }
}
```

뭔가 if 안에 for문까지는 괜찮은데 그 안에 다시 if문이 오는 게 복잡하다고 생각하여 다른 사람의 풀이를 보았고,
class Solution {
    public int solution(int n) {
            int answer = 0;
        if (n % 2 == 0) {
            for (int i = 2; i <= n; i += 2) {
                answer += i * i;
            }
        } else {
            for (int i = 1; i <= n; i += 2) {
                answer += i;
            }
        }
        return answer;
    }
}
이 코드처럼 내부 홀짝 판단 if문을 for문 안 조건문으로 처리하는 방법을 다음에는 직접 생각해낼 수 있으면 좋겠다고 생각함.
