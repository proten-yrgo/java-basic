<!-- 그림이나 예시를 들어 설명하면 나중에 복습할 때 좋습니다 -->
두 문자열 ineq와 eq로 계산을 해야하는데 어떻게 해야할지 감이 잡히지 않아 접근방법에 대해 검색하여
연산자 조합을 만들고 if문을 작성하였음. 그리고 맞으면 1 아니면 0 을 보고 boolean으로 접근하고자 생각했는데,
참, 거짓이 아니라 그에 맞는 숫자를 반환해야하는 것이므로 삼항연산자를 통해서 숫자로 배열하게끔 만듦.

## 💻 코드

```java
// 코드를 여기에 붙여넣어주세요 (주석 포함 권장)
class Solution {
    public int solution(String ineq, String eq, int n, int m) {
        String op = ineq + eq;
        boolean result = true;
        
        if (op.equals(">=")){
            result = n >= m;
        } else if (op.equals("<=")) {
            result = n <= m;
        } else if (op.equals(">!")){
            result = n > m;
        } else if (op.equals("<!")){
            result = n < m;
        }
        
        return result ? 1 : 0;
    }
}
```
